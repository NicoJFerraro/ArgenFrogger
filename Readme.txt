# ArgenFrogger

ArgenFrogger es un proyecto que realice para mi cursada de C++ para videojuegos. Es una reversión motivada por el Profesor Federico Olive de Image Campus, donde me motivó a darle un toque diferente al clásico juego 2D de la rana.

Es un juego que realice para Image Campus <a href="https://www.imagecampus.edu.ar/">Image Campus</a>
